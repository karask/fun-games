# How to run the Arcade Hub

Since the Arcade Hub uses modules and iframes to load games, it needs to be run on a local web server to avoid CORS (Cross-Origin Resource Sharing) issues that occur when simply opening the `index.html` file directly from your file system.

Here are a few quick ways to run it:

### Option 1: Using Python (Recommended if you have Python installed)
1. Open your terminal in the `fun-games` directory.
2. Run the following command:
   python3 -m http.server 8000
   (or `python -m http.server 8000` on Windows)
3. Open your browser and go to: http://localhost:8000

### Option 2: Using Node.js (If you have npm installed)
1. Open your terminal in the `fun-games` directory.
2. Run the following command:
   npx serve .
3. Open your browser and go to the URL provided in the terminal (usually http://localhost:3000)

### Option 3: Using VS Code (If you use VS Code)
1. Install the "Live Server" extension by Ritwick Dey.
2. Open the `fun-games` folder in VS Code.
3. Right-click on `index.html` and select "Open with Live Server" or click "Go Live" in the bottom right corner.
